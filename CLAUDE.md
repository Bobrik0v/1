# 3D-JET.by — сайт мастерской 3D-печати

Одностраничный статический сайт мастерской 3D-печати в Минске (Беларусь).
Живёт на **https://3d-jet.by/** (домен привязан к GitHub Pages в настройках репозитория).

## Структура

- `index.html` — вся страница: шапка, hero, услуги, оборудование, материалы (аккордеон), «как работаем», цены, контакты с формой, подвал с реквизитами ИП
- `css/style.css` — все стили (тёмный «премиальный» дизайн)
- `js/main.js` — мобильное меню, аккордеон материалов, отправка формы
- `robots.txt`, `sitemap.xml` — для поисковиков (адреса указывают на https://3d-jet.by/)
- `.github/workflows/pages.yml` — автодеплой на GitHub Pages

## Деплой — важно

Сайт публикуется **только из ветки `main`**: пуш в `main` запускает workflow «Deploy to GitHub Pages». Изменения в других ветках на сайт не попадают, пока их не вольют в `main`. После слияния проверяй, что запуск workflow завершился со статусом success.
