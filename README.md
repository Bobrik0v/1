# Сайт для бизнеса

Одностраничный лендинг: услуги, преимущества, тарифы и форма заявки.
Чистый HTML/CSS/JS — без сборки и зависимостей.

## Структура

- `index.html` — вся разметка страницы (тексты меняются здесь)
- `css/style.css` — стили (основной цвет — переменная `--accent` в начале файла)
- `js/main.js` — мобильное меню и обработка формы

## Как посмотреть

Просто откройте `index.html` в браузере, либо запустите локальный сервер:

```bash
python3 -m http.server 8000
```

и откройте http://localhost:8000

## Что заменить на своё

1. Название компании (`ВашаКомпания`) — в шапке, подвале и `<title>`
2. Тексты услуг, преимуществ и тарифов в `index.html`
3. Телефон, e-mail и адрес в секции «Контакты»
4. Форма сейчас только показывает сообщение об успехе — чтобы получать
   заявки, подключите [Formspree](https://formspree.io) или свой бэкенд
   в `js/main.js`
