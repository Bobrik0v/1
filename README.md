# 3D-JET.by — сайт мастерской 3D-печати в Минске

Одностраничный лендинг: 3D-печать, сканирование, моделирование, ремонт
мелкой бытовой техники (скоро), гид по материалам, цены и форма заявки.
Чистый HTML/CSS/JS — без сборки и зависимостей.

## Структура

- `index.html` — вся разметка страницы (тексты меняются здесь)
- `css/style.css` — стили (основной цвет — переменная `--accent` в начале файла)
- `js/main.js` — мобильное меню и обработка формы

## Как посмотреть

Просто откройте `index.html` в браузере, либо запустите локальный сервер:

```bash
python3 -m http.server 8000
```

и откройте http://localhost:8000

## Что осталось доделать

1. Цены на печать и моделирование в секции «Цены» — сейчас ориентировочные
2. Форма отправляет заявки на почту через [FormSubmit](https://formsubmit.co)
   (адрес задан в `js/main.js`) — после первой отправки нужно один раз
   подтвердить почту по письму от FormSubmit
